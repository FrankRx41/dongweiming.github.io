    命令调度模式

    设想一个问题，假如你经常搞web框架，你知道GET/POST/PUT/DELETE，当你想确定使用那种方式工作你需要大概这样:

    if command == 'get':
    get()
elif command == 'put':
    put()
else:
    error()

    抑或写一个字典作为调度的方式：

    dispatch_table = {
    'get': get,
    'put': put,
}

if dispatch_table.has_key(command):
    func = dispatch_table[command]
    func()
else:
    error()

    其实呢？python已经自带了很多这样的功能的东西，Guido van Rossum把这个模式叫做Command Dispatch
