    if command == 'get':
    get()
elif command == 'put':
    put()
else:
    error()

    抑或写一个字典作为调度的方式：

    dispatch_table = {
    'get': get,
    'put': put,
}

if dispatch_table.has_key(command):
    func = dispatch_table[command]
    func()
else:
    error()
