工厂方法模式

这是第二种'工厂'模式，Factorymethod和简单工厂的区别我想是： 简单工厂是用工厂类的不同方法返回不同产品类实例， 工厂方法根据工厂类的一个方法的不同参数返回不同产品类实例, 但是这些产品类一般都有共同的父类

python的例子

class Cup:
    color = ""

    # 这里我用类方法实现工厂方法 
    @staticmethod
    def getCup(cupColor):   
        if (cupColor == "red"):
            return RedCup()
        elif (cupColor == "blue"):
            return BlueCup()
        else:
            return None

class RedCup(Cup):
    color = "red"

class BlueCup(Cup):
    color = "blue"

这样测试一下吧

# 给这个共用的方法传入参数'red'
redCup = Cup.getCup("red")
# 这里返回RedCup类的实例
print "%s(%s)" % (redCup.color, redCup.__class__.__name__)

blueCup = Cup.getCup("blue")
print "%s(%s)" % (blueCup.color, blueCup.__class__.__name__)
